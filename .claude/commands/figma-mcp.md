# /figma-mcp

Use `skills/figma-portfolio-replica/SKILL.md`.

Activate this workflow when the task is to:

- use or troubleshoot Figma MCP
- translate a Figma link into code for this repository
- implement or update a portfolio section from a Figma node

Preferred inputs:

- Figma URL with `node-id`
- section name or page area affected
- any constraints on fidelity, responsiveness, or animation

Expected flow:

1. Read the skill file.
2. Use Figma MCP tools if available.
3. Inspect the existing section in the repo.
4. Implement with the repo's Next.js, TypeScript, Tailwind, and motion conventions.
5. Verify with `/verify-app` when code changed.
